## Installation

For the UI:
`cd planes-ui` <br/>
`npm install --force` - to install all packages<br/>

For the server:
`cd planes-server` <br/>
`npm install` - to install all packages

## Starting

For the UI:
`cd planes-ui` <br/>
`npm start`

For the server:
`cd planes-server` <br/>
`npm start`
