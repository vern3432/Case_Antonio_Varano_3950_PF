## Installation

For the UI: <br/>
`cd planes-ui` <br/>
`npm install --force` - to install all packages<br/>

For the server: <br/>
`cd planes-server` <br/>
`npm install` - to install all packages

For the database: <br/>
`cd planes-server` <br/>
`node script.js` - to create the database

## Starting

For the UI: <br/>
`cd planes-ui` <br/>
`npm start`

For the server: <br/>
`cd planes-server` <br/>
`npm start`
